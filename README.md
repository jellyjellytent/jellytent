# Jellytent

> AI-powered video chat agent for JellyJelly

## Overview

Jellytent is a real-time video chat agent that provides an interactive jellyfish avatar companion. Built for integration with the JellyJelly chat platform.

## Status

🚧 **Early Development** - Not ready for production use.

## Getting Started

```bash
npm install
npm run dev
```
